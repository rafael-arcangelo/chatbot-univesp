### Como usar este projeto?

- Clone ou baixe este repositório
- Entre na pasta do repositório
- Execute `npm install`
- Execute `npm run start:dev`
- Abra seu navegador e digite a URL: `http://localhost:8000`
- Faça a leitura do QR Code
